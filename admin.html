How Auto-Indexing Works

      

        

          
STEP 01

          
Push an HTML file

          
Upload any .html file to your GitHub repo on the main branch.

        

        

          
STEP 02

          
GitHub Action triggers

          
The workflow in .github/workflows/sync-pages.yml runs automatically.

        

        

          
STEP 03

          
pages.json updates

          
The script scans for new files, adds them with auto-detected title & description.

        

        

          
STEP 04

          
Index refreshes

          
index.html reads pages.json on load — your page appears automatically.

File structure

      
your-repo/
├── .github/workflows/sync-pages.yml   ← auto-runs on push
├── scripts/sync-pages.js              ← scanner & updater
├── pages.json                         ← source of truth
├── index.html                         ← public index
└── admin.html                         ← this page
