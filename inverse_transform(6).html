The Problem: Your Computer Only Knows One Trick

Imagine you're a bouncer at a club. People walk in randomly. You need to split them in exactly the right proportions:

10%VIP
60%Dance Floor
30%Bar

You pull out a spinner (0–100) and send each person based on where it lands.

This IS the inverse transform method.

Your computer's only random trick:
rand() → uniform number in [0,1]

Every value between 0 and 1 is equally likely. That's it.
What you actually want:
Numbers that follow a specific shape — bell curve, ramp, exponential... any distribution you choose.

The Central Question of This Lecture

Given U ~ Uniform(0,1), find transformation g
such that Y = g(U) follows any distribution FX

The CDF: Your Secret Weapon

What is the CDF doing?

The CDF F_X(x) answers: "What fraction of values are ≤ x?"

It always outputs a number between 0 and 1.

It always goes from 0 to 1, left to right.

F_X maps: x → [0, 1]
F_X⁻¹ maps: [0, 1] → x

The inverse CDF flips the machine backwards. And what else lives in [0,1]? Our uniform U!

The Graphical Method — Move the slider to read off F_X⁻¹(u)

0.50 → F⁻¹(u) = 0.00

The Proof — Step by Step

Goal: Show that if Y = F_X⁻¹(U), then Y has distribution F_X

Click each step to reveal it on the board:

Click Step 1 below to start...
1
Write the CDF of Y
Start with the definition of CDF for our new variable Y
2
Substitute Y = F_X⁻¹(U)
Replace Y with our transformation
3
Apply F_X to both sides
F_X is monotone, so we can apply it to the inequality
4
Use U ~ Uniform(0,1)
For Uniform, P[U ≤ t] = t. That's the key!

Why Step 3 is Valid — The "Log Analogy"

Think of it like taking log of both sides of an inequality:

If a ≤ b, then log(a) ≤ log(b)
← because log is monotone increasing

If F_X⁻¹(U) ≤ y, then
F_X(F_X⁻¹(U)) ≤ F_X(y)
     U ≤ F_X(y)

Worked Examples — By Hand

Ramp (Simple)
Exponential
Gaussian

f_X(x) = 2x, for 0 ≤ x ≤ 1

A ramp shape. Values near 1 are more likely.

Step 1: Find the CDF
F_X(x) = ∫₀ˣ 2t dt = x²

Step 2: Invert
u = x² → x = √u

∴ F_X⁻¹(u) = √u

Try it live:

0.64
Y = √0.64 = 0.80

f_X(x) = λe^(-λx), for x ≥ 0

Exponential distribution. Used for modeling wait times.

Step 1: Find the CDF
F_X(x) = 1 − e^(−λx)

Step 2: Set u = F_X(x) and solve for x
u = 1 − e^(−λx)
e^(−λx) = 1 − u
−λx = ln(1 − u)

∴ F_X⁻¹(u) = −(1/λ)·ln(1−u)
1.0
0.50
Y = −(1/λ)·ln(1−U) = 0.693

X ~ Gaussian(μ, σ²)

No closed form for Φ⁻¹, but computers have it built in.

F_X(x) = Φ((x−μ)/σ)

Invert:
F_X⁻¹(u) = σ·Φ⁻¹(u) + μ

Φ = standard normal CDF
Φ⁻¹ = quantile function
0.0
1.0
0.84
Y = σ·Φ⁻¹(U) + μ = 1.00
